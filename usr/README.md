# City Runner 3D

A modern 3D-style endless running mobile game with boy and girl characters, smooth animations, and fun arcade gameplay built in Flutter.

## Features
- Endless running mode with increasing difficulty
- Swipe controls (left/right to switch lanes, up to jump, down to slide)
- Collect coins and dodge obstacles
- Pseudo-3D perspective rendering
- Main Menu with character selection
- Game Over screen with score tracking

## Tech Stack
- Flutter
- Dart
- CustomPainter for 3D perspective

## Setup
1. Run `flutter pub get`
2. Run `flutter run`
