# Vision AI Studio

Vision AI Studio is a futuristic AI-powered mobile application designed for generating ultra HD 4K images and cinematic videos using Artificial Intelligence. With a premium glassmorphism design and dark neon theme, it offers a professional creation platform directly on your device.

## Features

- **AI Image Generator**: Create stunning 4K images in various styles (Anime, Realistic, Cinematic, Fantasy).
- **AI Video Generator**: Convert text to cinematic videos and create animations with motion effects.
- **AI Photo Enhancement**: Upscale images, remove blur, and restore old photos.
- **AI Editing Tools**: Apply AI filters, beauty modes, and create avatars.
- **AI Prompt Assistant**: Get smart suggestions and use voice-to-prompt features.
- **Export & Social**: Export in multiple aspect ratios (Reels, YouTube Shorts) and share directly.
- **Premium Subscription**: Unlock faster rendering, unlimited generations, and exclusive models.

## Tech Stack

- **Framework**: Flutter
- **UI Design**: Modern glassmorphism with neon accents
- **Cross-Platform**: Supports iOS, Android, and Web

## How to Run

1. Ensure Flutter is installed.
2. Clone the repository and run `flutter pub get`.
3. Launch the app using `flutter run`.
